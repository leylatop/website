介绍：使用id 和 use 元素，可以重用元素。
  1. 将id 赋给需要被重用的元素
  2. 使用use 元素，href 属性指向被重用的元素
  3. use 元素可以使用transform 属性，对被重用的元素进行变换
draw a sun

1. draw a circle and a line

2. add id and use, to reuse the line. draw the sun

draw a snowflake

1. draw a branch

2. draw sub branches

3. add id and use, to reuse the branch. draw the snowflake
