总结

  1. circle 元素的cx和cy是圆心的坐标，r是圆的半径
  2. rect 元素的x 和 y是矩形左上角的坐标，rx是矩形圆角的大小
  3. line 元素的x1y1、x2y2是结对使用的，表示一条直线的起点和终点
  4. polygon 元素的points由多个点组成，顺时针连接，一个点由x,y组成，点与点之间由空格隔开，会自动封闭；经常用于画多边形
  5. polyline 元素的points由多个点组成，顺时针连接，一个点由x,y组成，点与点之间由空格隔开，不会自动封闭；经常用于画折线
  6. css 的 stroke-linecap，可以设置line、polyline两端为圆角
  7. css 的 stroke-width，可以设置line、polyline的宽度，xy坐标位于line的中间
